| 2PL           | 자회사 개념의 물류 자체 회사내에 있던 물류부서를 독립법인화 하여 이 독립법인화된 업체에 물류를 위탁하는 형태의 물류                                                                                                                                                                                                                                                                  |                                                                                      |
| 3PL           | 물류 아웃소싱의 형태. 아웃소싱 보다 포괄적인 의미이며 포괄적인 아웃소싱에는 2PL과 3PL, 4PL이 모두 포함되며 3PL의 개념은 2PL을 벗어나서 물류 전문업체에게 물류를 전담시키는 형태를 말한다.                                                                                                                                                                                            |                                                                                      |
| 4PL           | 최근 일부에서 주장되는 것으로 단순한 물류의 전담만이 아니라 물류 컨설팅, IT화 된 시스템을 가지고 물류에 대한 통합적인 서비스를 하는 업태라고는 하지만 3PL과 경계가 모호함.                                                                                                                                                                                                           |                                                                                      |
